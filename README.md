# r5n-labs Homebrew Tap

Homebrew formulae for r5n-labs projects.

## Installation

```bash
brew tap r5n-labs/tap
```

## Available

### vigil

```bash
brew install r5n-labs/tap/vigil
```

| Platform | Architecture |
|----------|-------------|
| macOS | arm64, x64 |
| Linux | arm64, x64 |

### invoicy

```bash
brew install --cask r5n-labs/tap/invoicy
```

| Platform | Architecture |
|----------|-------------|
| macOS | arm64 |
| Windows | x64 |
| Linux | x64 |

Windows and Linux installers available at [releases.r5n.dev](https://releases.r5n.dev).

## Updating

```bash
brew update
brew upgrade
```
